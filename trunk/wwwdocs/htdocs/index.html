 What is GPUTILS ?
GPUTILS is a collection of tools for the Microchip (TM) PIC microcontrollers.  It includes gpasm, gplink, and gplib. It's distributed under the terms of the GNU Public License. See the gnupic page for other GNU PIC tools such as gpsim, a gputils compatible simulator.
Who is GPUTILS ?
Currently, it is primarily supported by Craig Franklin <craigfranklin@attbi.com> and Scott Dattalo <scott@dattalo.com>.
A Brief History
In 1997,  James Bowman <jamesbowman@users.sourceforge.net> started gpasm.  The objective was to create an open source replacement for Microchip's MPASM.  During this time many people contributed to gpasm.
    Scott Dattalo <scott@dattalo.com>
    James Cameron <james.cameron@digital.com>
    Stefan Petersen <spe@stacken.kth.se>
    Eric Smith <eric@brouhaha.com>
    Alessandro Zummo <azummo@ita.flashnet.it>
    Andrew Burgess <aab@cichlid.com>
    Bruce Stough <sbs@wavefront.com>
    Tommy Thorn <thorn@acm.org>
    Charlie Krauter <charlie@mother.com>
At the end of 2000, Craig Franklin <craigfranklin@attbi.com> became the primary gpasm developer.  Through most of 2000 the focus was compatiblity with MPASM.  This was made much easiler by a contribution of MPASM test files to the project by Microchip.

In 2001, work started on relocatable object tools for gpasm.  A new package, gputils, containing gpasm and the new tools was created.
 
What Platforms are Supported ?
Linux (and other unices) and MS-Windows are supported. GPUTILS will compile natively on Linux using gcc. Windows 98/NT users will need MinGW32.
Downloading GPUTILS
The GPUTILS source tree is currently located at SourceForge.

A release of GPUTILS has not been made.  Please see gpasm for a current release.

Work continues in CVS. If you have CVS capability, the following will get you the latest sources.

cvs -d:pserver:anonymous@cvs.gputils.sourceforge.net:/cvsroot/gputils login

The password is empty and pressing [ENTER] will get you in. 

The following command will checkout all of the sources and place them in a gpasm subdirectory.

cvs -z3 -d:pserver:anonymous@cvs.gputils.sourceforge.net:/cvsroot/gputils co gputils

You can browse our CVS tree here. 

SourceForge has further documentation on accessing a CVS tree here.

A new linker and lib tool are under development.  These new tools along with gpasm will be distributed in a new package gputils, GNU PIC Utilities.
